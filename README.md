# iComputer

#### 描述
该项目是一款查看电脑CPU和显卡性能天梯图，还可以查看电脑配件组合后价格的小程序

#### 安装
1. 克隆项目
2. 执行npm install 命令
3. 使用微信开发者工具进行预览
